# IBKR 报表精简分析工具

移动端优先的网页工具，用于上传 IBKR Activity Statement PDF，并生成账户总览与股票/期权综合收益排行。

线上地址：

```text
https://duxin100100.github.io/-/
```

## iPhone 离线安装

1. 用 iPhone Safari 打开上面的 HTTPS 地址。
2. 等页面显示“离线缓存已准备好，可添加到主屏幕。”
3. 点 Safari 分享按钮。
4. 选择“添加到主屏幕”。
5. 之后从主屏幕图标打开，断网也能上传 PDF 并解析。

注意：首次打开需要联网加载并缓存 PDF 解析组件；缓存完成后可离线使用。

## 功能

- 上传 `.pdf` 报表
- 解析账户总览：累积存款、结束价值、累计收益、收益率
- 解析股票收益：已实现盈亏、未实现盈亏
- 解析期权已实现收益，并按底层 Symbol 合并
- 按总盈亏从高到低排序
- 移动端卡片展示
- 导出面板截图 PNG
- PWA 离线缓存，可添加到 iPhone 主屏幕

## 数据口径

期权收益默认只统计期权已实现盈亏，不把期权未实现盈亏混入。金额支持逗号与括号负数格式，例如 `(1,234.56)` 会识别为 `-1234.56`。
